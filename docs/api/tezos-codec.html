Usage

  • tezos-codec [global options] command [command options]
  • tezos-codec --help (for global options)
  • tezos-codec [global options] command --help (for command options)
  • tezos-codec --version (for version information)

To browse the documentation

  • tezos-codec [global options] man (for a list of commands)
  • tezos-codec [global options] man -v 3 (for the full manual)

Global options (must come before the command)

-d --base-dir <path>: data directory The directory where the Tezos codec will output logs. By default: '$HOME/.tezos-client'.

Access the documentation

  • man [keyword...] [-v --verbosity <0|1|2|3>] [--format <plain|colors|html>]
    Print documentation of commands. Add search keywords to narrow list. Will display only the commands by default, unless [-verbosity <2|3>] is passed or the list of matching commands if less than 3. keyword: keyword to search for If several are given they must all appear in the command. -v --verbosity <0|1|2|3>: level of details 0. Only shows command mnemonics, without documentation. 1. Shows command mnemonics with short descriptions. 2. Show commands and arguments with short descriptions 3. Show everything --format <plain|colors|html>: the manual's output format Defaults to `plain`.

Commands to handle encodings

  • list encodings
    List the registered encoding in Tezos.
  • dump encodings [--compact]
    Dump a json description of all registered encodings. --compact: Output json descriptions without extraneous whitespace characters
  • encode id from json
    Encode the given JSON data into binary using the provided encoding identifier. id: Encoding identifier json: JSON file or data
  • decode id from hex
    Decode the binary encoded data into JSON using the provided encoding identifier. id: Encoding identifier hex: Binary encoded data
  • display id from binary hex
    Display the binary encoded data using the provided encoding identifier. id: Encoding identifier hex: Binary encoded data
  • display id from json json
    Display the JSON encoded data using the provided encoding identifier. id: Encoding identifier json: JSON file or data
  • describe id binary schema
    Describe the binary schema associated to the provided encoding identifier. id: Encoding identifier
  • describe id json schema
    Describe the JSON schema associated to the provided encoding identifier. id: Encoding identifier
  • slice hex
    Attempts to slice an hex-encoded binary value with all known encodings. hex: Binary encoded data
  • slice hex with encoding id [--format <FORMAT>]
    Slice an hex-encoded binary value with the specified encoding. hex: Binary encoded data id: Encoding identifier --format <FORMAT>: The format to print the output in: json, pretty, or waterfall. Defaults to `pretty`.
